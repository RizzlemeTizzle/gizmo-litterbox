# Persona: De Esthetisch Bewuste Kattenouder

## Snapshot
- **Name:** Laura (or Mark) van Dijk
- **Age:** 32 (range 28–40)
- **Gender split:** ~70% female, 30% male
- **Location:** Urban NL/BE (Rotterdam, Amsterdam, Utrecht, Antwerp)
- **Household:** Cohabiting, maybe young kids; 1–2 cats treated as family
- **Home:** Apartment / row house with modern, curated interior
- **Income:** €45k–70k per year
- **Education:** HBO/WO
- **Professions:** Creative/knowledge work (marketing, design, IT, consultancy, healthcare, education)

## Psychographics & Lifestyle
- **Values:** Aesthetics, cleanliness, sustainability, wellbeing (human + pet)
- **Belief:** Functional design should also be beautiful—utility can be elegant
- **Behavior:** Research-heavy, reads reviews, compares options, price-aware but quality-first
- **Price attitude:** Will pay premium for durable, stylish, problem-solving products
- **Channels:** Instagram, Pinterest, curated Facebook groups for cats/interiors
- **Style cues:** Scandinavian, Japandi, modern boho inspiration
- **Associated brands:** IKEA (design utility), Dyson (premium tech), Rituals (sensory calm), Litter-Robot, Catit, Bol.com premium

## Needs & Pain Points
| Need | Description |
| --- | --- |
| Odor + leakage control | Apartment living means any smell lingers; old boxes stained floors |
| Easy maintenance | Wants to clean in seconds without scooping sludge |
| Cat comfort | Spacious, non-claustrophobic, suits cats that spray/stand |
| Interior fit | Refuses to tolerate ugly plastic tubs in living spaces |
| Sustainability | Prefers wood pellets, low-waste materials |

## Psychological stage (Breakthrough Advertising)
- **Stage:** Solution-aware. Knows odor/design solutions exist, but unsure which one solves everything.
- **Goal:** Prove GIZMO uniquely blends design + hygiene + cat comfort.

## Key purchase drivers
1. **Aesthetic win:** “Eindelijk een kattenbak die je interieur niet verpest.”
2. **Hygiene confidence:** “No smell, no leaks, thanks to the two-layer pellet system.”
3. **Cat wellbeing:** “Jouw kat voelt zich relaxed in een ruime, open box.”
4. **Ease:** “Schoon in seconden, zonder vieze handen.”

## Objections & responses
| Objection | Response angle |
| --- | --- |
| “Het is duur voor een kattenbak.” | Highlight longevity, odor savings, interior upgrade; compare to designer furniture pieces |
| “Past het wel in mijn kleinere appartement?” | Use lifestyle shots showing footprint + storage ideas; mention Essentials/smaller versions |
| “Werkt dit als mijn kat geen houtkorrels wil?” | Explain optional double bottom removal + compatibility with clumping litter |

## Motivating content
- Before/after apartment photos showing clean setups.
- Short reels of cleaning process (seconds, no smell).
- Real reviews emphasizing leak-free experience.
- Storylines about balancing design life with pet love (“mooi huis, blije kat”).

## Campaign triggers
- Social proof from design influencers.
- UGC of cats enjoying the box.
- Emails referencing common frustrations (“Ben je klaar met lekken en stank?”).

Use this persona to shape copy, ad creative, and wholesale pitches aimed at design-forward pet owners.
