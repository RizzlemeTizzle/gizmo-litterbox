# Product Overview

## Core value props
- **Leak-free, odor-controlled design:** Two-layer sieve system optimized for wood pellets dramatically cuts ammonia smell.
- **Interior-friendly aesthetics:** Minimalist silhouette, matte finish, available in white/black variants that blend into modern homes.
- **Large format comfort:** 58 × 40 × 40.5 cm footprint suits bigger cats and those that spray/stand while urinating.
- **Easy maintenance:** Flip-top access, integrated scoop + bags, compatible with both pellet and clumping litter (double bottom optional).

## Technical specs
| Attribute | Details |

| Compatible litter | Wood pellets by default; works with clumping if insert removed |

## Customer feedback themes (Bol.com reviews)
- **Pros:** “Mooi design”, “lekvrij”, “geen stank”, “kat vindt het fijn”, “makkelijk schoonmaken”.
- **Cons / objections:** Large footprint for small apartments, premium price perception, some cats dislike pellet texture (double bottom unused).
